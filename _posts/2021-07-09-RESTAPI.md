<ul>
    <li> Find processes running on all ports </li>  
</ul>

```
sudo lsof -i -P -n | grep LISTEN

```

<ul>
    <li> Kill process running on a particular port (say, 5000) </li>  
</ul>

```
kill -9 PID 5000

```
